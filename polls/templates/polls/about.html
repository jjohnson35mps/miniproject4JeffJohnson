What this site does
  • CFP Champion Predictor: Users pick who they think will win the 2025 College Football Playoff, using a list of top-ranked teams.
  • Stored Votes: Every pick is saved in the database using the ChampionshipPick model.
  • Live Leaderboard: We total all picks and display vote counts and percentages in real time.
  • Classic Polls: Fans can vote on stuff like best rivalry, loudest stadium, best tailgate food, etc. Each poll tracks votes per choice.
  • Admin Tools: Django admin lets us browse and manage picks, questions, and choices.
  • Auth-ready: Login is supported for admin, and the app is structured so we could tie one pick per user in the future.
Tech stack
  • Django 5.x for views, models, URL routing, and templates
  • SQLite for persistent storage / migrations
  • Bootstrap 5 + custom theme in static/polls/style.css
  • Django Admin for data entry and moderation
