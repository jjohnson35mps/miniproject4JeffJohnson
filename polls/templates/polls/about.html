About This Project

This Django web app is my Mini Project Four.

  • Users predict the 2025 College Football Playoff champion by picking from the AP Top 25.
  • Votes are stored in the database (ChampionshipPick model).
  • Results are aggregated and shown with live percentages.
  • Admin interface lets us browse and filter all submitted picks.
  • Site uses Bootstrap styling, modals, and a POST form.
  • Login is supported for admin; the app is ready for per-user vote tracking.

Tech stack: Django 5.x, SQLite, Bootstrap.
